# NEXTIER Platform - OutreachGlobal.io

> Agnostic SMS outreach platform. Works for any deal type.

## What It Is

NEXTIER is an SMS outreach machine. Import leads, enrich them, send campaigns, handle replies, book meetings, close deals. Works for any business that needs to reach people.

**Infrastructure:** SignalHouse.io (10DLC approved, compliant)

## The Machine

```
IMPORT → ENRICH → SEND → REPLY → BOOK → CLOSE → REPEAT
```

| Step | What Happens |
|------|--------------|
| Import | CSV, API, B2B search - get your leads in |
| Enrich | LUCI finds real phone numbers ($0.05/lead) |
| Send | GIANNA opens with your message |
| Reply | Leads respond, agents handle it |
| Book | SABRINA schedules the meeting |
| Close | Deal moves through pipeline |
| Repeat | Pull the lever again |

## Pipeline Stages

```
DISCOVERY → QUALIFICATION → PROPOSAL → NEGOTIATION → CLOSED
```
