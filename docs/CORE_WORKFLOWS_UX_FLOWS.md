# API QUICK REFERENCE

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/api/datalake/list` | GET | List bucket contents by prefix |
| `/api/datalake/query` | POST | Query records with filters |
| `/api/datalake/sector-upload` | POST | Import CSV to sector |
| `/api/enrichment/skip-trace` | POST | Single skip trace lookup |
| `/api/enrichment/bulk-skip-trace` | POST | Batch skip trace |
| `/api/webhook/skip-trace` | POST | Receive skip trace webhooks |
| `/api/signalhouse` | GET | Check SignalHouse status |
| `/api/signalhouse` | POST | Send SMS, provision number, verify phone |
| `/api/campaign/push` | POST | Create campaign with leads |
| `/api/webhook/signalhouse` | POST | Receive SMS webhooks |
| `/api/gianna/voice-webhook` | POST | Handle inbound calls |

# KEY METRICS TO TRACK

| Metric | Description | Target |
|--------|-------------|--------|
| **Skip Trace Hit Rate** | % of lookups returning mobile | >60% |
| **Campaign-Ready Rate** | % of enriched with mobile | >50% |
| **SMS Delivery Rate** | % delivered to carrier | >95% |
| **Response Rate** | % leads who reply | >5% |
| **Positive Intent Rate** | % positive replies | >30% of responses |
| **Opt-Out Rate** | % who reply STOP | <3% |
